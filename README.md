# Discord Moderation Bot
Basic discord moderation bot.

#### Commands:
* ban
* kick
* unban
* purge
* nuke
* addrole
* removerole
* mute
* tempmute
* unmute
* help
* ping

#### Config:

```json5
{
  "botToken": "",//The discord bot token.
  "prefix": "!", //The prefix for the bot commands.
}
```
